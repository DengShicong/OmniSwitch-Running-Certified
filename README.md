# OmniSwitch-Running-Certified
通过show running-directory查看交换机是否重启未保存配置并将回显信息收集只保留关键部分<br>
并生成html文件，在html文件修改字符颜色。<br>
当Running configuration: CERTIFIED,时显示红色。<br>
当 Running configuration: WORKING,时显示绿色<br>
